STEPS TO REPRODUCE RESULTS

1. Open ML_FINAL_PROJECT.R in RStudio.
2. Navigate to SEQUENTIAL LOOP.
3. Change value of path variable to locate data files.
4. Change loop indexes to include as many events as you wish to process.
5. Run the loop.
6. Event scores will be printed in the terminal, and saved in the scores data frame.
